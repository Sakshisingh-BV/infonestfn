## 📊 **Metrics & Specifications**

### Design System
```
Color Palette:
  Primary Gradient:    #667eea → #764ba2 (135deg)
  Text Primary:        #1a202c
  Text Secondary:      #4a5568
  Text Muted:          #718096
  Backgrounds:         rgba(255, 255, 255, 0.95)
  Borders:             rgba(102, 126, 234, 0.1-0.3)

Typography:
  Font Family:         Inter, -apple-system, BlinkMacSystemFont, Roboto
  Headings:            700 bold, 1.5-2rem
  Labels:              600 bold, 0.875-1rem
  Body:                500 normal, 0.875-1rem

Spacing Scale:
  xs: 0.5rem    (small gaps)
  sm: 0.75rem   (button padding top)
  md: 1rem      (card padding)
  lg: 1.5rem    (section gaps)
  xl: 2rem      (major padding)

Radius Scale:
  sm: 8px       (small elements)
  md: 12px      (buttons)
  lg: 16px      (tabs, cards)
  xl: 20px      (headers)
```

### Animation Specifications
```
Transitions:    0.3s cubic-bezier(0.4, 0, 0.2, 1)
Shimmer:        3-4s ease-in-out infinite
Rotation:       20s linear infinite
Scale Range:    1.0 → 1.08 (8% enlargement)
Translate Y:    3px lift on hover, 1px on active
Opacity:        0-1 on page entry

Easing Function: Material Design standard
  cubic-bezier(0.4, 0, 0.2, 1)
  Fast start, smooth deceleration
```

### Shadow System
```
Light (default):      0 4px 12px rgba(0, 0, 0, 0.1)
Medium (hover):       0 8px 25px rgba(0, 0, 0, 0.15)
Strong (focus):       0 20px 40px rgba(0, 0, 0, 0.2)
Glow (accent):        0 0 0 1px rgba(255, 255, 255, 0.2)
Inner (polish):       inset 0 1px 0 rgba(255, 255, 255, 0.1)
Calendar (tile):      0 4px 12px rgba(102, 126, 234, 0.2)
```
